# TKiK

Repozytorium z materiałami i projektami na zajęcia TKiK (AGH).

## Projekty

### `kompilator/` — Kompilator Pascal → C

Edukacyjny kompilator tłumaczący podzbiór Pascala do C.

- Kod i instrukcja: `kompilator/README.md`
- Szybki start (z root repo):

```bash
python3 kompilator/main.py kompilator/test.pas kompilator/test.c
gcc kompilator/test.c -o kompilator/test
./kompilator/test
```
